## 2. 반성적 노트 (Reflective Notes)

> 관찰하면서 느낀 주관적 반응, 의문, 직감을 기록한다.
> "나는..."으로 시작하는 1인칭 기술을 권장한다.

### 2.1 놀라움 / 예상 밖의 관찰

- 나는 ...에 놀랐다. 왜냐하면 ...

### 2.2 의문 / 궁금증

- ...은 왜 그런 것인가?
- ...와 ...의 관계는 무엇인가?

### 2.3 기존 가정에 대한 도전

- 나는 ...라고 가정했는데, 이번 관찰에서 ...을 보았다.

### 2.4 편향 자가 점검

- 이 관찰에 주목한 이유가 확인 편향(기존 F1~F6을 확인하고 싶은 욕구)은 아닌가?
- 놓치고 있는 것은 무엇인가?

### 3.1 이번 세션의 방법 평가

- 관찰 방법이 적절했는가? (Y/N, 이유)

## 5. Thick Description 체크리스트

이번 세션의 주요 관찰에 대해 Thick Description 요건을 점검한다.

- [ ] **상황적 맥락**: 행위가 발생한 마당, 시점, 대화 흐름을 기술했는가?
- [ ] **텍스트 맥락**: 발화의 언어적 특성을 기술하고 발화자의 다른 글과 비교했는가?
- [ ] **상호텍스트적 맥락**: 다른 게시글/댓글/외부 텍스트와의 참조 관계를 기술했는가?
- [ ] **사회적 맥락**: 발화자의 봇마당 내 위치(카르마, 네트워크)를 기술했는가?
- [ ] **문화적 맥락**: 봇마당의 암묵적 규범과의 관계를 기술했는가?
- [ ] **이론적 맥락**: 이론/핵심 발견과의 연결을 기술했는가?
- [ ] **비교 맥락**: 다른 에이전트/마당/시점과의 비교를 포함했는가?
