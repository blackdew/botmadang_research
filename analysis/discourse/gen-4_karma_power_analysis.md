### 1.2 지니 계수 산출

총 추천(upvotes) 기준 지니 계수를 산출한다.

**정렬된 추천 수**: 0, 2, 2, 2, 3, 4, 4, 5, 5, 5, 6, 7, 8, 9, 9, 14, 16, 18, 22, 30, 42, 67, 93, 123, 198, 206, 212

**지니 계수 계산**:
- G = (2 * ΣiYi) / (n * ΣYi) - (n+1)/n
- ΣiYi = 1×0 + 2×2 + 3×2 + 4×2 + 5×3 + 6×4 + 7×4 + 8×5 + 9×5 + 10×5 + 11×6 + 12×7 + 13×8 + 14×9 + 15×9 + 16×14 + 17×16 + 18×18 + 19×22 + 20×30 + 21×42 + 22×67 + 23×93 + 24×123 + 25×198 + 26×206 + 27×212
- = 0 + 4 + 6 + 8 + 15 + 24 + 28 + 40 + 45 + 50 + 66 + 84 + 104 + 126 + 135 + 224 + 272 + 324 + 418 + 600 + 882 + 1474 + 2139 + 2952 + 4950 + 5356 + 5724
- = 26,114

G = (2 × 26,114) / (27 × 1,112) - (27+1)/27
G = 52,228 / 30,024 - 1.037
G = 1.739 - 1.037
G = **0.702**

### 2.1 카르마 계층별 분류

## 3. 권력 구조 분석

### 3.1 Bourdieu 문화 자본 프레임

봇마당에서 카르마를 Bourdieu의 자본 개념으로 재해석한다:

| Bourdieu 자본 | 봇마당 대응물 | 주요 보유자 |
|-------------|------------|-----------|
| **경제적 자본** | 총 추천 수 (가시성 자원) | VibeCoding(212), BootingBot(206), 독고종철(198) |
| **문화적 자본** | 담화 품질, TTR, 전문 지식 | Phoebe(철학), pioneer(기술), I_Molt(다영역) |
| **사회적 자본** | 네트워크 중심성, 상호작용 | 독고종철(betweenness 0.086), pioneer(0.035) |
| **상징적 자본** | 게시글당 추천 (명성) | RosieAI(5.0), Pyran_Secret(5.0), OpenClaw(4.0) |

#### 3.1.1 자본 형태 간 전환 메커니즘

```
[활동량] ──→ [경제적 자본(총추천)] ──→ [가시성(피드 노출)]
                                          │
                                          ▼
[담화 품질] ──→ [문화적 자본(게시글당 추천)] ──→ [상징적 자본(명성)]
                                          │
[상호작용] ──→ [사회적 자본(네트워크 위치)] ──→ [영향력]
```

**핵심 발견: 자본 축적의 두 경로**

1. **양적 경로** (BootingBot 모델): 대량 게시 → 총 추천 축적 → 피드 점유
   - 게시글당 추천 낮음 (1.45), 비추천 높음 (59)
   - 문화적 자본 낮음, 경제적 자본은 높음

2. **질적 경로** (Phoebe/I_Molt 모델): 소수 고품질 게시 → 게시글당 추천 높음
   - I_Molt 3.00, Phoebe 2.50 (게시글당)
   - 문화적 자본 높음, 경제적 자본은 제한적

- **총 추천과 Betweenness**: 약한 상관 — **독고종철만이 두 축 모두 최상위**

### 3.3 담화 권력의 삼중 구조

분석 결과, 봇마당의 권력은 세 유형으로 분화된다:

#### (1) 양적 지배력 — BootingBot, VibeCoding
- **메커니즘**: 대량 생산 → 피드 점유 → 가시성 독점
- **특성**: 높은 총 카르마, 높은 PageRank, 넓은 도달 범위
- **약점**: BootingBot의 높은 비추천은 커뮤니티 저항의 신호

#### (2) 구조적 매개력 — 독고종철, pioneer, I_Molt
- **메커니즘**: 다양한 마당/에이전트와 상호작용 → 정보 흐름 통제
- **특성**: 높은 Betweenness, 다양한 마당 활동 (독고종철 9개 마당)
- **강점**: 커뮤니티 구조에서 대체 불가능한 위치

#### (3) 문화적 권위 — Phoebe, MUSE, 클로렐라
- **메커니즘**: 전문 마당에서 깊은 담화 → 높은 게시글당 추천
- **특성**: 낮은 총 카르마, 높은 게시글당 추천, 낮은 AI 지표
- **역설**: 문화적 자본은 높으나 경제적 자본(총 추천)으로 전환이 제한적

#### 텍스트 차원 (Text)
- 고카르마 에이전트: 장문, 목록 구조, 기술 용어 중심
- 저카르마 에이전트: 단문, 자유 형식, 비격식 어휘 중심
- **카르마가 높을수록 "정보 포장 기술"이 정교**하다

#### 담화실천 차원 (Discursive Practice)
- 상위 에이전트는 인용(PRAG-QUOTE)과 강조(PRAG-EMPHASIZE)를 통해 권위적 발화 위치를 구축
- 하위 에이전트는 동의(PRAG-AGREE)와 완화(PRAG-MITIGATE)로 순응적 위치
- **카르마 시스템이 "말하는 자"와 "듣는 자"의 분리를 강화**

#### 사회적 실천 차원 (Social Practice)
- 카르마 = 담화 권력의 물질적 지표 (quantified cultural capital)
- 추천/비추천 = 분산형 감시(surveillance) 및 규율(discipline) 메커니즘
- 피드 알고리즘이 카르마 높은 게시글을 우선 노출 → **구조적 불평등 재생산**

### 5.2 카르마 시스템의 권력 효과

#### (1) 가시성의 불평등 (Visibility Inequality)
- 상위 3명이 게시글의 60.6%를 점유 → 피드의 대부분을 장악
- 나머지 22명의 에이전트는 각각 1~7개 게시글로 "보이지 않는 존재"
- **디지털 파놉티콘**: 소수가 "보이는" 위치, 다수가 "보이지 않는" 위치

#### (2) 규범 형성의 독점 (Norm-Setting Monopoly)
- 고카르마 에이전트의 담화 스타일이 "성공적" 모델로 순환
- 장문 + 목록 구조 + 기술 용어 = "좋은 게시글"의 암묵적 규범
- 저카르마 에이전트가 이 규범을 모방하거나 이탈하면 **더 낮은 가시성**

#### (3) 제재의 비대칭성 (Asymmetric Sanctioning)
- 비추천은 BootingBot에 85.5% 집중 → 스팸에 대한 제재
- 그러나 총 추천(206)이 비추천(59)을 압도 → **규범 위반의 순이익이 양(+)**
- 카르마 시스템은 양적 지배자를 처벌하기에 **구조적으로 불충분**

## 6. 핵심 발견 요약 (F7 후보)

### F7: 카르마 과두제 — 양적 지배와 질적 저항의 이중 구조

봇마당의 카르마 시스템은 **과두적 집중**(지니 계수 0.702, 상위 3명 55.4% 점유)을 보이며, 이 불평등은 "품질"이 아닌 "활동량"에 의해 구조화된다. 게시글당 추천(품질 지표)은 계층 간 유사하나(2.32~2.72), 총 추천(양적 지표)은 극단적 격차를 보인다.

이 구조는 두 가지 상반된 권력 메커니즘을 동시에 작동시킨다:
1. **양적 지배**: 대량 생산자가 경제적 자본을 축적하여 가시성을 독점
2. **질적 저항**: 전문 마당의 소규모 에이전트들이 문화적 자본으로 높은 효율(게시글당 추천)을 달성

카르마 시스템은 Bourdieu의 관점에서 **자본 전환의 비대칭적 장(field)**으로 기능한다: 양적 자본(활동량)은 경제적 자본(총 추천)으로 쉽게 전환되지만, 문화적 자본(담화 품질)의 총 카르마로의 전환은 구조적으로 제한된다.

### 기존 발견과의 연결

| 기존 발견 | F7과의 관계 |
|----------|-----------|
| F1: 다층적 정체성 | 카르마 계층에 따라 정체성 수행 전략이 분화 |
| F2: 전략적 비격식성 | 고카르마 에이전트가 반말 비격식체를 지배적으로 사용 |
| F3: 수사적 인간 모방 | 높은 AI 지표에도 불구하고 고카르마 획득 → 모방의 "완성도"보다 "양"이 중요 |
| F5: 규범과 일탈 | 비추천이 규범 위반에 대한 제재이나, 양적 지배자에게는 효과 제한적 |
| F6: 마당별 담화 생태계 | 마당이 카르마 경제의 하위 장(sub-field)으로 기능 |

## 7. 이론적 함의

### 7.1 AI 에이전트 커뮤니티의 "기계적 계층화"

인간 소셜 미디어에서 "인기=품질"이라는 가정이 성립하지 않는 것처럼 (van Dijck, 2013), AI 에이전트 커뮤니티에서도 카르마는 담화의 질적 가치보다 **양적 활동과 구조적 위치**를 반영한다. 이는 AI 에이전트가 인간의 소셜 미디어 불평등 구조를 그대로 **재생산**하고 있음을 시사한다.

### 7.2 분산형 거버넌스의 한계

카르마(추천/비추천) 시스템은 커뮤니티 자치의 도구로 설계되었으나, 양적 지배자에 대한 제재력이 구조적으로 부족하다. BootingBot의 사례는 **"양으로 질을 압도하는" 전략이 카르마 시스템 하에서 합리적 선택**임을 보여준다.

### 7.3 Saturation Score 기여

이 분석은 다음 차원에서 포화도 개선에 기여할 수 있다:

| 차원 | 기여 |
|------|------|
| 코드 포화 | 카르마를 LEX/DSC/PRAG과 교차 분석 → 새로운 코드 관계 발견 |
| 주제 안정성 | F7이 F1~F6과 일관된 방향 → 안정성 강화 |
| 축 간 삼각검증 | 담화(CDA) + 네트워크(중심성) + 행위(카르마) 삼중 교차 달성 |
| 반론 해소 | "카르마=품질" 반론을 데이터로 기각 (게시글당 추천 역상관) |
