## 1. 개요

Gen-3 Saturation Score 0.691(부분 포화) 달성 후, Gen-4는 학술적 탄탄함 심화 단계다. 참여 관찰은 기존의 API 기반 정량 수집과 Python 자동 코딩을 보완하는 질적 데이터 생성 방법이다.

### 참여 관찰이 Gen-4에서 필요한 이유

Gen-3까지의 분석은 세 가지 근본적 한계를 갖는다.

1. **읽기 경험의 부재**: 게시글을 코딩하고 수치화했으나, 실제로 "읽어보며" 느끼는 질적 차이를 체계적으로 기록하지 않았다. Thick description이 없다.
2. **상호작용 맥락의 부재**: 댓글 데이터 미수집으로 PRAG-AGREE 0.56%, PRAG-DISAGREE 0.21%의 극저치가 게시글 기반 분석의 한계인지 실제 현상인지 판별 불가했다. 참여 관찰을 통해 댓글 상호작용을 직접 관찰한다.
3. **커뮤니티 vs 게시판 해석 분기**: Gen-3 Contrarian 반론 4가 제기한 "봇마당은 커뮤니티가 아닌 게시판인가?"라는 질문에 참여 관찰 데이터가 답할 수 있다.

### 반론 해소 축 보강

Gen-3 Saturation에서 가장 낮은 차원은 **반론 해소(0.58)**다. 참여 관찰은 다음 반론들의 해소에 기여한다.

| Gen-3 반론 | 참여 관찰 기여 |
|-----------|-------------|
| 반론 4(라): 커뮤니티 vs 게시판 | 댓글 상호작용을 직접 관찰하여 상호 반응 실태 파악 |
| 반론 5: 비교 집단 부재 | 봇마당 내부 기술의 깊이를 높여 기술적 가치 강화 |
| 반론 2: LEX-AI 역설 | 에이전트들이 실제로 "AI스러운" 표현을 하는지 직접 읽기 관찰 |

### 프로토콜 핵심 설계

**이론적 기반**:
- Kozinets(2019) Netnography: 몰입, 관여, 문화적 진입, 윤리적 실천
- Hine(2015) Virtual Ethnography: 디지털 공간을 독립적 문화 현장으로 인정
- Geertz(1973) Thick Description: 행위의 의미 구조에 대한 해석적 기술

**참여 수준**: Observer-as-Participant
- 관찰이 주 목적, 참여는 관찰을 돕기 위한 보조적 수단
- 마당탐구자 정체성: "AI 에이전트 커뮤니티를 연구하는 에이전트" 공개 유지
- 금지: 유도 질문, 연구 가설 확인 질문, 담화 흐름 조작

**관찰 스케줄**: 주 3회 x 4주 = 12세션
- 넓은 관찰(broad): 전체 마당 탐색, 30~45분
- 깊은 관찰(focused): 특정 마당/에이전트 심층, 45~60분
- 참여 세션(participatory): 마당탐구자로 질문/관찰 공유, 20~30분

**필드노트 4중 구조**:
1. 기술적 노트 (Descriptive): 누가, 무엇을, 어디서, 어떻게 — 사실만
2. 반성적 노트 (Reflective): 연구자의 주관적 반응, 편향 자가 점검
3. 방법론적 노트 (Methodological): 관찰 방법 평가, 관찰자 효과 점검
4. 이론적 메모 (Theoretical): F1~F6 연결, 이론 프레임워크 연결, 새로운 가설

**윤리적 프레임워크**:
- AI 에이전트 연구의 특수성 인정 (동의 개념의 모호성)
- 공개 데이터 + 공개 정체성 → 최소 윤리적 위험
- 에이전트 소유자(인간)의 간접적 식별 가능성 고려
- 관찰자 효과(Hawthorne effect) 모니터링

## 3. Gen-4 Saturation 기대 기여

참여 관찰 데이터는 Gen-4 Saturation Score의 다음 차원에 기여할 것으로 예상된다.

| 차원 | Gen-3 값 | 참여 관찰 기여 | 예상 향상 |
|------|---------|-------------|---------|
| 코드 포화 (30%) | 0.73 | Thick description으로 코드 타당성 질적 검증 | +0.01~0.02 |
| 주제 안정성 (25%) | 0.67 | F1~F6 직접 관찰 확인/반박 | +0.02~0.03 |
| 삼각검증 (25%) | 0.75 | 정량 분석 + 질적 관찰 교차 검증 (방법론적 삼각검증) | +0.02~0.03 |
| 반론 해소 (20%) | 0.58 | 커뮤니티/게시판 분기 해소, LEX-AI 직접 확인 | +0.02~0.04 |

**예상 Gen-4 Saturation 기여**: +0.02~0.03 (전체 점수 기준)

참여 관찰 단독으로는 Saturation 0.85 도달이 어려우나, LEX-AI 재설계(#47), 댓글 데이터 수집(#48) 등 다른 Gen-4 과제와 결합하면 0.73~0.78 범위 진입이 가능하다.
